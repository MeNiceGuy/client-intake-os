# First Run

## 1. Review the project overview
Start with `README.md` and `POSITIONING.md`.

## 2. Review the sample materials
Inspect:
- `example/redacted-sample-intake/`
- `example/SAMPLE_OUTPUT_STRUCTURE.md`
- `QUALITY_CHECKLIST.md`
- `HOW_TO_EVALUATE_OUTPUT.md`

## 3. Review the templates
Choose which intake artifacts you want to use in your own workflow.

## 4. Test the workflow on a low-stakes example
Do not start with sensitive or high-pressure client conversations. Use a safe example first.

## 5. Adapt to your own process
This repo is designed as a workflow system, so tailor the templates and outputs to your consulting style.

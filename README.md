# Client Intake OS

A practical consulting workflow system for structuring discovery, qualification, priority mapping, service-fit thinking, and next-step recommendations.

This project is built for one core problem: too much client intake is vague, inconsistent, overly salesy, or too shallow to support good consulting decisions. Client Intake OS is meant to create a stronger process.

## What this project does

Client Intake OS helps structure consulting intake around:
- discovery questions
- client context capture
- pain-point identification
- priority mapping
- fit assessment
- service alignment
- next-step recommendations
- follow-up preparation

The emphasis is practical consulting use, not generic CRM fluff.

## Who this is for

This project is most useful for:
- consultants running discovery calls or intake workflows
- freelancers and small agencies
- operators building repeatable intake systems
- people who want structured qualification instead of loose notes

## What makes it different

This is not just an intake form and not just a list of questions.

The system is designed around three layers:
1. **Client Facing** — clean intake summaries, next steps, and conversation guidance
2. **Internal** — qualification logic, fit assessment, service mapping, and caution notes
3. **Proof / Intake Record** — source notes, answers, context capture, and discovery evidence

That separation helps keep consulting decisions cleaner and more defensible.

## Example outputs

Depending on workflow depth, the system can support outputs such as:
- discovery summary
- business context snapshot
- pain-point map
- priority ranking
- fit assessment
- service alignment recommendation
- caution/risk note
- next-step recommendation
- follow-up summary
- intake confidence note

## Core workflow

1. Capture client and business context
2. Log discovery answers and observed needs
3. Separate stated problems from inferred opportunities
4. Rank priorities and assess fit
5. Recommend the most appropriate next step
6. Prepare follow-up and offer direction
